# memwise - Memory wisely works

## Support containers
1. Dynamic array (array_t)
2. Temporary array (tempo_t)
3. Hash table (table_t)
4. Queue (queue_t)
5. Stack (use array_t)

## Support memory buffers (like allocator)
1. Linear
2. Stack
3. Global/System heap
4. Double end stack
5. Pool
